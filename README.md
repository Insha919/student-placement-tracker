# Student Placement Tracker

A web application built using Python, Flask and MySQL to simplify the college placement process.

## Features

- Student Registration
- Student Login Authentication
- Dashboard
- Company Management
- Eligibility Checking
- Application Tracking

## Tech Stack

- Python
- Flask
- MySQL
- HTML
- CSS

## Future Enhancements

- Resume Upload
- Email Notifications
- Placement Analytics
- Admin Dashboard
